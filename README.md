# 🌌 平行人生 · Life Decision OS

> **在一个决定之前，先活一遍未来**
>
> 基于蒙特卡洛模拟 × 行为经济学的通用人生决策操作系统 · MVP Demo

---

## 🎯 这是什么？

**平行人生** 不是传统意义上的"职业规划工具"或"决策建议系统"——它是一个**通用决策操作系统**，通过模拟你在不同选择下的"平行时空人生"，让你**看见每一条路的尽头有什么在等你**。

### 核心理念

> 市面上的工具告诉你"你应该选 A"，
> 而平行人生告诉你——"这是选 A 和选 B 之后，你的人生可能的样子。你自己选。"

### 本 Demo 首发场景

**职业转型**：留在国企 vs 出去创业 — 这是一个 29 岁国企工程师面对的真实困境。

## 🔑 技术亮点

### 1. 叙事个性化「B+C 组合」
- **B 方案（原词回注）**：从用户输入中抽取"自由/女儿/父母"等原词，强制嵌入叙事
  - 你写 → *"自由对我最重要"*
  - 系统生成 → *"你说过，**自由**对你意味着一切。第 5 年，你终于明白——**自由**从来不是免费的…"*
- **C 方案（LLM 升级路径）**：`core.py` 顶部 `USE_LLM=False` 开关 + `call_llm()` stub，可无缝接入 GPT-4/Claude

### 2. 蒙特卡洛 · 对数正态 + 失败事件
```python
income = np.random.lognormal(log_mu, log_sigma, N=1000)
if failure_prob > 0.1:
    failed = np.random.random(N) < cumulative_fail
    income[failed] = uniform(8, 20)   # 创业失败残值
```

### 3. 三维索引叙事模板
```
NARRATIVES[选项][主价值维度][结局档][年份]
   = 2 × 5 × 3 × 4 = 120 槽位（已填 60+）
```

### 4. 认知偏差四字段卡片
不只是「提示」，而是「对话」：
- **证据**：你在描述中提到了「老王」
- **解释**：可得性启发是用容易想起的案例代替真实概率
- **反思一问**：你身边那个创业成功的人只是样本。同批次出发的 100 个人里，现在在哪里？

## 🛣️ 升级路线图

**Phase 1 → Phase 2（接入真实 LLM）**
```python
# core.py 顶部
USE_LLM = True

# call_llm() 填入真实调用
import openai
return openai.ChatCompletion.create(model="gpt-4", ...)
```

**Phase 2 → Phase 3（新增场景）**
- 复制 `SCENARIOS["career_transition"]` → 填充新场景参数
- 在 `NARRATIVES` 里为新选项 id 新增叙事模板
- 在 `SCENARIO_CATALOG` 把 `enabled` 设为 `True`
